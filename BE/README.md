## Back end to a shopping website

To run:

```
npm install
npm start
```

To run in development mode:

```
npm install
npm run dev
```

To change environment variables, edit `.env` file:

-   `PORT=<your-port>`
-   `URI=<your-db-uri>`
-   `ADMIN_PASSOWRD=<admin-password>`
-   `FIREBASE_SERVICE_ACCOUNT_JSON=<path/to/json>`
-   `BUCKET_ID=<bucket-id>`
